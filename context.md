## Website Structure

### Tech Stack
- Pure HTML / CSS / JS — no frameworks, no build tools
- Fonts: Syne (display), Inter (body), JetBrains Mono (mono) via Google Fonts
- Color scheme: Dark bg `#0A0A0F`, Red accent `#FF3C28`
- Hosted on GitHub Pages

### File Structure
```
servedhot-site/
├── index.html          — Full single-page site
├── styles.css          — Cinematic dark theme (~3000 lines)
├── script.js           — Interactions, video controls, accordion, particles
├── context.md          — This file
├── .gitignore          — .claude, .DS_Store excluded
├── assets/
│   ├── logos/           — 33 brand logo PNGs + servedhot-logo.jpg
│   ├── images/          — nescafe.png, jummy.png, pizza-hut.png, cinepolis.png, bazaari.png
│   ├── video/           — served-hot-video.mp4, IQOO.mp4, kewal-masala.mp4, zenma.mp4, tanya-mitra.mp4
│   └── og-image.jpg     — 1200x630 social sharing image
```

### Page Sections (in order)

1. **Splash Screen** — Cinematic intro with film grain, letterbox bars, typing "PRODUCTIONS", auto-dismiss ~4.2s, skip button
2. **Navigation** — Sticky header with logo (servedhot-logo.jpg), links: Services, Brands, About, Get In Touch. Active link highlights on scroll.
3. **Hero** — "We Serve It Hot." headline + video showreel (served-hot-video.mp4) with mute/unmute button. Video starts after splash. Showreel links to Instagram reel.
4. **Work/Portfolio** — HIDDEN (display:none). 11 cards with videos, YouTube thumbnails, and local images. Ready to unhide when needed.
5. **Services** — Bento grid with 9 cards. Each card has background Unsplash image, dark gradient overlay, hover zoom + color reveal. Two hero cards (End-to-End + Post Production), one wide card (Product Videography), rest standard.
6. **About (Who We Are)** — Editorial split layout. Video of Tanya Mitra (60% width, 4:5 aspect), bio text + 2x2 stats grid on right. Founder badge links to LinkedIn.
7. **Brands** — Floating logo constellation (33 logos). Organic layout, no boxes. Logos float with gentle animation, grayscale default, full color + glow halo on hover. Header has "collective experience" transparency note.
8. **Process (How We Work)** — 4-step timeline: Discovery, Creative Dev, Production, Post-Production.
9. **Testimonials** — 3 client quotes with star ratings.
10. **CTA Marquee** — "LET'S CREATE SOMETHING HOT" scrolling text.
11. **Contact** — No form. Giant clickable email (tanya.mitra97@gmail.com) as centerpiece, phone, location, social icons (Instagram + LinkedIn linked, Behance + YouTube placeholder).
12. **Footer** — Logo, tagline, nav links, service links, social links, copyright 2025.
